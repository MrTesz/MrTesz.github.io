# Features des Floose SMP

Unser Smp bietet viele Features wie ein [Auktionshaus](auctions.html), um seine Gegenstände an andere Spieler zu verkaufen, ein [Job System](jobsystem.html), um auch ohne Handel Coins sammeln zu können, einen [Seasonpass](seasonpass.html) um beim Spielen nebenbei nützliche Gegenstände zu sammeln und vieles mehr.
